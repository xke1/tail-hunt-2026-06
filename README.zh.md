# tail-hunt — 一张"读得懂世界"的彩票

🌐 [English](./README.md) | **中文**

> 小资金期权,凸性优先,赌右尾。

---

## 这是什么

**期权也是一种赌博——没人知道未来会发生什么。** 但相比买彩票,它更有参与感和互动性:你得先读懂世界正在发生什么,才下得了注。

我开源了一个专门做这个的系统:**多 agent、多重否定**,从三个维度的交集来判断一个机会:

1. **这标的本身会不会暴力波动?** —— 绝对脆弱
2. **premium 还便不便宜?** —— 凸性还便宜吗
3. **这个风险,市场反应得怎么样了?** —— 是不是已经被消化/共识

三个都中,才告诉你今天有值得开仓的机会。**就算没有开仓机会,也只要在 workspace 里说一句"从矮子里拔高个",AI 就会推荐你一些标的,值得你去交易所(IBKR)细致地看一看。**

欢迎大家探讨、挑刺、证伪。**不构成任何投资建议,尤其针对大资金。**

---

## 老实说的核心

老实说,这更像一张 **不知道有没有正 EV 的彩票**,不是什么 alpha 机器。我不吹"我有 edge"。它的价值是:固定时间,**逼你去读懂世界、有参与感、和市场互动**——你为这点参与感付的,就是那一点点小亏。刮刮乐让你什么都学不到、毫无参与感;这个"彩票"让你看懂世界在崩什么。

## 一些设计细节

- **逻辑主导、数字验证。** 脆弱是**推理**出来的,不是**测量**出来的。你不能用 IV 算出"这东西会崩",只能用因果链论证它会崩,再用数字看保险贵不贵。
- **独立 skeptic agent——不是自己批改自己作业。** 每个候选由一个**独立**的 agent 攻击,它唯一的任务就是用带来源的硬证据杀掉这个想法;杀不死才留下。(至今:**6/6 个想法全被杀。**)
- **5 条防自欺铁律**写进 skill:① 顺序锁(先纯逻辑,禁碰数字)② 推理链留痕(≥5步因果链,禁直接给结论)③ 证伪前置(提候选前先写它怎么死)④ 真对抗螺旋(具体杀招、每轮新角度、吵到杀不动)⑤ 要"能被杀死的推理",不是"写得长"(把烂想法包装得很深是更高级的自欺)。
- **无论有没有单,都给你一张地图:** 全域脆弱局势盘(8域)+ 分类侦察清单(农产品/利率/能源/金属/股票/加密…)+ **死路博物馆**(每个被杀的想法 + 杀招——*"为什么没下注"才是最有用的部分*)。
- **三个已经用真金白银换来的教训:**(a)*真的不挤吗?*(做空长债自以为非共识,其实是最挤的交易之一)(b)*载体对吗?*(ETF 的 contango/错配能吃光你的 edge——WEAT 不是那个真紧的小麦;UNG 十年跌 ~89%)(c)*催化剂方向对吗?*
- **数据:** 免费 OpenBB(~15分钟延迟、IV 不准)。**执行:** IBKR,小资金,**只买或定义风险价差,禁裸卖**(无限亏损绝对禁止)。
- **大多数时候它说"今天没有"——这是特性不是 bug。** 拥挤的故事保险早就贵了,能拒绝出手才是它最值钱的地方。

## repo 里有什么

- `.claude/skills/tail-hunt/` — **核心**:`SKILL.md`(怎么想)+ `OUTPUT_TEMPLATE.md`(输出长啥样)+ 防自欺规矩。(英文版:`SKILL.en.md` / `OUTPUT_TEMPLATE.en.md`)
- `rank.py` — 排行榜引擎(~110 标的:实现波动/IV/IV-RV/skew,~2分钟)。
- `deep_read.py` — 单标的整张期权表深读(逐 strike IV/RV/skew + 幻觉检查)。
- `tail_scanner.py` / `hist_analog.py` / `credit_drill.py` / `hyg_ladder.py` — 数据探针。
- `ibkr_bridge.py` — IBKR 只读桥(`ib_async`)。
- `output/` — 每次推理的完整记录(带时间戳,含死路)。

## 怎么跑

```bash
python3.12 -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
export FRED_API_KEY=你的免费key   # https://fredaccount.stlouisfed.org/apikeys
python rank.py            # 排行榜
python deep_read.py HYG   # 单标的深读
```
OpenBB 接进 Claude Code 的 MCP 配置见 `.mcp.json`。skill 在 `.claude/skills/tail-hunt/`。
